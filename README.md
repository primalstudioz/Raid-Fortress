# Raid Fortress

uses Source code for Source SDK 2013.

Contains the game code for Half-Life 2, HL2: DM and TF2.

Raid mode is a cancelled and scrapped gamemode from Team-Fortress 2. That was a prototype PVE gamemode. That had many of its elements recycled for Mann Vs Machine. 
Gameplay is very similar to Left4Dead. Although including many differences with boss battles at the end of every map.
